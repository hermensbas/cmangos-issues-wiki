h3. Description of the fields

h4. entry

The ID to identify the text.
To avoid conflicts between Expansions, please use:
* Vanilla
2000000000 and 2000009999
* TBC
2000010000 and 2000019999
* WotLK
2000020000 +

h4. content&#95;default

The text for default language. 
CmangoS provides English as default.
